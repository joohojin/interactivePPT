# 한여름밤의 조작된 사랑 - 인터랙티브 발표 웹

원본 PPT/PDF를 기반으로 만든 정적 HTML 발표 프로젝트입니다.

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다.

## 조작

- `←` / `→`: 이전/다음 슬라이드
- `Space` / `Enter`: 다음 슬라이드
- `Home` / `End`: 처음/끝 슬라이드
- `F` 또는 상단 `전체화면`: 전체화면 발표
- 상단 `☰`: 슬라이드 목차
- 상단 `노트`: 발표 노트

## 구조

- `index.html`: 발표 화면
- `css/styles.css`: 원본 스타일 기반 반응형 디자인
- `js/slides.js`: 슬라이드 원문/구조 데이터
- `js/app.js`: 슬라이드 전환, 전체화면, 목차, 인터랙션
- `assets/img`: PPTX에서 추출한 원본 이미지 자산
